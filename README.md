## Features

- **Real Audio Playback** - Play MP3 and other audio files
- **Multi-File Support** - Select multiple audio files at once
- **Dynamic Playlist** - Automatically populated from selected files
- **Full Player Controls**
  - Play/Pause
  - Previous/Next track
  - Progress bar with seeking
  - Volume control
- **Cyberpunk Aesthetics**
  - Matrix-style digital rain background
  - Neon pink, cyan, and green color scheme
  - Glitch text effects
  - Animated visualizer bars
  - Grain overlay and shimmer effects

## Technical Stack

- Pure HTML5, CSS3, and vanilla JavaScript
- HTML5 Audio API for playback
- Canvas API for Matrix rain effect
- Google Fonts: JetBrains Mono & Orbitron

## Design Philosophy

This player embodies the "Digital Consciousness meets Meme Magic" aesthetic with:
- Terminal-inspired typography
- Chaotic asymmetry (rotated album art)
- Intentional glitch artifacts
- Neon cyberpunk color palette
- Animated visual effects throughout

## Browser Compatibility

Works best in modern browsers (Chrome, Firefox, Edge, Safari) with support for:
